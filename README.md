# 无线触控板（Windows）

将手机浏览器变成电脑无线触控板，支持鼠标控制、快捷输入、媒体键和文件传输。

## 功能特性

- 触摸板控制：单指移动、双指滚动
- 左右键分区点击：左侧 65% / 右侧 35%
- 悬浮快捷按钮：输入、快捷键、媒体/文件控制
- 文件传输：手机选文件上传到电脑
- 上传目录可配置：在连接页点击“选择目录”并持久保存
- 系统托盘常驻：可从托盘快速打开连接页或退出

## 运行环境

- Windows 10/11
- Python 3.11+（建议使用项目内虚拟环境）

## 快速开始

```bash
# 安装依赖
uv pip install -r requirements.txt

# 启动
uv run main.py
```

启动后会自动打开连接页：

- `http://<电脑IP>:5000/connect`（电脑端扫码页）
- 手机扫码后进入触控板页并建立控制连接

## 使用说明

1. 手机与电脑连接同一局域网（同一 WiFi）。
2. 电脑运行程序后，打开连接页二维码。
3. 手机扫码进入触控板页面开始控制。
4. 如需改文件保存目录，在连接页使用“选择目录”。

## 主要交互

- 单指滑动：移动鼠标
- 单击触摸板左侧区域：鼠标左键
- 单击触摸板右侧区域：鼠标右键
- 双指滑动：滚轮滚动
- 底部按钮：ESC / 回车 / 删除
- 悬浮面板：文本输入、方向键、复制粘贴、媒体控制、文件上传


## 打包（仅窗口版）

```bash
python build.py
```

生成文件：

- `dist/无线触控板.exe`

## 常见问题

- 手机连不上：确认手机与电脑在同一网络，并检查防火墙是否放行 5000/4000 端口。
- 页面看起来没更新：强制刷新或清理浏览器缓存后重试。
- 文件上传后看不到：检查连接页设置的上传目录是否正确。
